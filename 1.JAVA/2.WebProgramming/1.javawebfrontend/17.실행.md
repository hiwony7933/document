## 자바 웹 프로젝트의 실행
1.eclipse에서 실행
* 프로젝트가 자신의 워크스페이스의 

\.metadata\.plugins\org.eclipse.wst.server.core\tmp0\wtpwebapps 디렉토리에 프로젝트가 복사되서 실행

실행이 되는 프로젝트는 복사된 프로젝트이지 소스 코드를 작성하고 있는 프로젝트가 아닙니다.

.이 붙는 디렉토리는 UNIX(Mac OS X), Linux에서는 숨김 디렉토리입니다.

Mac 에서 단축키가 shift + command + . 을 누르면 숨김 디렉토리가 보여집니다.

프로젝트에 어떤 파일을 만들어두고 읽을려고 했는데 실행을 할 때 실수로 만들지 않은 경우 프로젝트에 새로 만들어도 반영이 안되는 경우가 있습니다.

이런 경우에는 실행이 되는 프로젝트에 파일을 만들어 주면 됩니다.
